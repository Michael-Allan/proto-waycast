Michael Allan’s wayrepo

This is my wayrepo — a repository of files that encode my public expressions in regard to personal goals — a frame of self expression, a medium of interpersonal exchange, and the structural heart of a dispersive technique of communication that I call ‘waycasting’.

You can think of waycasting like a programmer might: it’s an interactive Greasemonkey for the ‘world computer’. The wayrepo is my user script and I can make it do anything I like.

If you haven’t experienced it yet, then the thing to watch in that metaphor is the interactive part: because this kind of user script (wayscript) runs not only in the objective world, but in the subjective, too, where its changes are felt immediately. So you won’t come out of it unchanged. But then, neither will the objective world.
